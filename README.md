# 婚活フェルミ推定

## 概要

婚活フェルミ推定は、理想の条件に合う未婚者が日本に何人くらいいそうかをフェルミ推定で可視化する静的 SPA です。結果の希少さや絞り込みの過程を一目で共有しやすくし、SNS で拡散されやすいバイラル体験を意図した構成にしています。

## 機能一覧

- 未婚人口を母集団にした婚活条件のフェルミ推定
- 年齢、性別、年収、資産、身長、学歴、非喫煙、薄毛でない、容姿の条件指定
- **年齢別確率テーブル**: 年収・資産・学歴・非喫煙・薄毛は年齢（および一部は性別）で通過率が変動。若年層ほど高収入率は低く、薄毛進行は少ないなど実態に近い推定
- **年収×資産の相関補正**: 独立仮定を緩めるため、両方選択時に資産通過率を1.5倍（上限1.0）に引き上げ
- 男女で切り替わる身長確率テーブル
- ヒーロー数字と希少度ステータスバッジ表示、バッジtier別のSNS投稿文言
- 絞り込み過程のブレークダウン表示
- URL クエリへの状態保存と復元
- X、LINE、Instagram 向け画像保存、スクリーンショット保存
- `html-to-image` を使った結果パネルの PNG 出力
- Next.js 静的エクスポート（`output: 'export'`）による `out/` 生成

## ローカル開発

```bash
npm install && npm run dev
```

ブラウザで `http://localhost:3000` を開いてください。dev server は **Turbopack** で動作します（webpack版で発生していたHMRクラッシュを回避するため）。本番ビルド (`npm run build`) は引き続き webpack で実行されます。

## テスト

```bash
npm test
```

Vitest で実行します。

## ビルド

```bash
npm run build
```

ビルド成功後に `out/` が生成されます。

## デプロイ

### A. 汎用WEBサーバー（Xserver 等）

`out/` 配下を丸ごとドキュメントルートにアップロードしてください。

### B. Vercel CLI

```bash
npm i -g vercel && vercel
```

対話式で設定し、数分で本番 URL を発行できます。

### C. Vercel GitHub連携

リポジトリを push したあと、Vercel ダッシュボードで Import してください。

## サブディレクトリ配置

`next.config.js` に `basePath: '/sub'` と `assetPrefix: '/sub/'` を追加してから再ビルドしてください。

## OGP画像差し替え

`public/og.png` を 1200×630 の画像に差し替えてから再ビルドしてください。

## 統計データ更新

`data/statistics.ts` を編集したあとに再ビルドしてください。

## 出典

- 国勢調査（令和2年） — 未婚人口
- 賃金構造基本統計調査 — 年収分布
- 国民健康・栄養調査 — 身長分布
- 家計の金融行動に関する世論調査 — 個人金融資産
- 学校基本調査・各大学募集要項 — 学歴分布
- 数値は概算であり、条件間の独立仮定のため実際の分布とは誤差があります

## 免責事項

本アプリはエンタメ目的の簡易推定ツールです。実際の婚活結果、個人属性、母集団分布を厳密に表すものではありません。
