# 婚活フェルミ推定

## 概要

婚活フェルミ推定は、理想の条件に合う未婚者が日本に何人くらいいそうかをフェルミ推定で可視化する静的 SPA です。結果の希少さや絞り込みの過程を一目で共有しやすくし、SNS で拡散されやすいバイラル体験を意図した構成にしています。

## 機能一覧

- 未婚人口を母集団にした婚活条件のフェルミ推定
- 年齢、性別、年収、資産、身長、学歴、非喫煙、薄毛でない、容姿の条件指定
- 男女で切り替わる身長確率テーブル
- ヒーロー数字と希少度ステータスバッジ表示
- 絞り込み過程のブレークダウン表示
- URL クエリへの状態保存と復元
- X、LINE、Instagram 向け画像保存、スクリーンショット保存、URL コピー
- `html-to-image` を使った結果パネルの PNG 出力
- Next.js 静的エクスポートによる `out/` 生成

## ローカル開発

```bash
npm install && npm run dev
```

ブラウザで `http://localhost:3000` を開いてください。

## テスト

```bash
npm test
```

Vitest で実行します。

## ビルド

```bash
npm run build
```

ビルド成功後に `out/` が生成されます。

## デプロイ

### A. 汎用WEBサーバー（Xserver 等）

`out/` 配下を丸ごとドキュメントルートにアップロードしてください。

### B. Vercel CLI

```bash
npm i -g vercel && vercel
```

対話式で設定し、数分で本番 URL を発行できます。

### C. Vercel GitHub連携

リポジトリを push したあと、Vercel ダッシュボードで Import してください。

## サブディレクトリ配置

`next.config.js` に `basePath: '/sub'` と `assetPrefix: '/sub/'` を追加してから再ビルドしてください。

## OGP画像差し替え

`public/og.png` を 1200×630 の画像に差し替えてから再ビルドしてください。

## 統計データ更新

`data/statistics.ts` を編集したあとに再ビルドしてください。

## 出典

- 国勢調査（令和2年） — 未婚人口
- 賃金構造基本統計調査 — 年収分布
- 国民健康・栄養調査 — 身長分布
- 家計の金融行動に関する世論調査 — 個人金融資産
- 学校基本調査・各大学募集要項 — 学歴分布
- 数値は概算であり、条件間の独立仮定のため実際の分布とは誤差があります

## 免責事項

本アプリはエンタメ目的の簡易推定ツールです。実際の婚活結果、個人属性、母集団分布を厳密に表すものではありません。
